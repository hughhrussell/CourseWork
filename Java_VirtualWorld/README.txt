

Virtual World Program

Written in Java for an object oriented course

The goal was a introduction to OOP.

It was designed to represent an individual and certain characteristics.
-I added a vehicle class to describe an automobile
-The shoutbox class was intended to create messages to be printed out
-Classes were created to test basic funcationality (MyCloneTest,ShoutBoxTest,VechicleTest)
-The Expression check class was to check data entry for fitting a regular expression
